# Лабораторная работа 2: NLP
## Епифанов Евгений Валерьевич М8О-412Б-22

### Задача:
1. Установить ollama
2. Скачать в ней Qwen2.5:0.5b
3. Поднять Ollama сервер с LLM
4. Написать скрипт, отсылающий запросы (через curl или requests) на поднятый ollama сервер
5. Прогнать скрипт через 10 произвольных запросов и зафиксировать вывод LLM
6. Составить отчет инференса с двумя столбцами: (1) запрос к LLM, (2) соответствующий вывод LLM

### Стек:
1. Ollama
2. Qwen2.5:0.5B

### Требования:
1. Инференс Qwen2.5:0.5B на сервере ollama, слушающий и отвечающий на запросы по HTTP
2. Скрипт, отправляющий запросы (curl или requests) на сервер ollama для LLM модели
3. Dosctring документация всех функций
4. Отчет инференса

### Инструкция к установке и запуску:
1. Скачайте ollama с официального сайта - https://ollama.com/download

2. Запустите сервер через `ollama serve` (в отдельном терминале)

3. Установите модель с помощью `ollama pull qwen2.5:0.5b`

4. Установите библиотеку `requests` через `pip install requests`

5. Запустите [скрипт](main.py) через `python main.py`

6. Будут последовательно выведены результаты запросов к модели
